AWS Academy — database backup folder
=====================================

seed-original.sql
  A copy of the full canonical seed (DROP/CREATE card + INSERT all rows).
  Do not edit this file for day-to-day site changes; treat it as a restore point.
  To restore the database from this backup:
    mysql -u root -p < database/backup/seed-original.sql

The live seed you edit or regenerate is database/seed.sql (and scripts/extract_seed.py).
Refresh seed-original.sql when you intentionally want to save a new baseline (copy seed.sql here).
